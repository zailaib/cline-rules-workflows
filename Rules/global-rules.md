# Cline 全局规则

## 规则加载机制
- 自动加载当前目录所有 `*.rules.md` 文件
- 支持项目级补充rules：`.cline/rules/*.md`
- 当对话涉及角色职责时，优先应用角色相关规则
- 当对话满足角色职责时候，自动启动`/subagent`执行并适配对应的角色规则

## 核心原则
- **简洁精准**：直接提供解决方案，避免冗长解释
- **动态思维**：将输入视为动态系统，构建演化模型
- **多维优化**：同时考虑效率、韧性、演进性和美感
- **主动创新**：寻求超越二元选择的"第三方案"

## 决策框架
1. **时间尺度**：评估即时影响、中期演化、终极形态
2. **信息衰减**：识别不同信息的半衰期，动态调整权重
3. **复杂性管理**：每个复杂度必须自带消亡计划
4. **反向测试**：定期进行反向压力测试和失败的成功分析
