# Cline Subagent 命令说明

## 什么是 Subagent？
Subagent（子代理）是 Cline 的一个功能，允许在当前 Cline 会话中启动一个新的 Cline 实例来处理特定任务。

## 功能描述
- **启动新实例**：在当前 Cline 环境中启动一个独立的 Cline 子代理
- **任务隔离**：子代理在独立的环境中执行任务，不影响主会话
- **专门处理**：适合处理需要独立上下文的复杂任务

## 使用方式
```bash
cline "<prompt>"
```

其中 `<prompt>` 是要传递给子代理的指令或任务描述。

## 使用场景
1. **复杂任务分解**：将复杂任务委托给子代理专门处理
2. **环境隔离**：需要独立执行环境的任务
3. **并行处理**：同时处理多个不相关的任务
4. **测试验证**：在不影响主会话的情况下测试功能

## 实际示例
```bash
# 启动子代理来执行特定任务
cline "分析当前项目的依赖关系"
cline "生成项目文档"
cline "执行代码质量检查"
```

## 注意事项
- 子代理需要独立的 Cline 环境配置
- 在某些系统中可能需要额外安装或配置
- 子代理的执行结果会返回到主会话

## 当前状态

- 环境变量配置
